## 🛠️ Kurulum

1. Projeyi klonlayın
```bash
git clone https://github.com/kullaniciadi/proje-ismi.git
```

2. Proje dizinine gidin
```bash
cd proje-ismi
```

3. Bağımlılıkları yükleyin
```bash
npm install
```

4. Projeyi başlatın
```bash
npm start
```
